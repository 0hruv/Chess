# Chess

A fully functional chess game built with vanilla JavaScript, HTML, and CSS. Play chess in your browser with complete piece movement validation, check/checkmate detection, and turn-based gameplay.

## Features Implemented

- ♟️ All 6 piece types with correct movement rules
- ✓ Check and checkmate detection
- ✓ Turn-based gameplay (White vs Black)
- ✓ Move validation and illegal move prevention
- ✓ Visual move indicators showing valid moves
- ✓ Capture mechanics

## Unimplemented Features

1. **En Passant** - Special pawn capture move
2. **Castling** - King and rook special move
3. **Pawn Promotion** - Upgrade pawns reaching the end
4. **Stalemate Detection** - Draw condition when no legal moves
5. **50-Move Rule** - Draw after 50 moves without capture/pawn move
6. **Threefold Repetition** - Draw after same position repeats 3 times
7. **Move History & Undo** - Replay and undo moves
8. **Game Reset** - Start new game without page refresh
